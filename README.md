**MagicStory** is an interactive application that generates personalized bedtime stories for children and narrates them with a soothing, lifelike neural voice.

Built for the **Google AI Hackathon**, this project leverages **Gemini 1.5 Flash** for creative storytelling and **Edge-TTS** for high-quality audio narration.

## üöÄ Features
* **Unique Stories:** Instantly generated based on the child's name, age, and a chosen theme.
* **Audio Narration:** Uses advanced neural text-to-speech for a human-like voice (no robotic sounds!).
* **Cozy Interface:** Custom "Night Mode" UI designed for bedtime reading.
* **Bilingual:** Fully supports both English and French.
